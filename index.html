Configuration

おすすめ設定 (.vimrc)

コピーしてすぐ使える設定例

基本設定

" 行番号表示
set number

" シンタックスハイライト
syntax on

" タブをスペースに
set expandtab
set tabstop=2
set shiftwidth=2

" 検索時にハイライト
set hlsearch

" 検索時に大文字小文字を区別しない
set ignorecase
set smartcase

見た目の改善

" カラースキーム
set background=dark
colorscheme desert

" カーソル行をハイライト
set cursorline

" 対応する括弧を表示
set showmatch

" ステータスラインを常に表示
set laststatus=2

" コマンドラインの高さ
set cmdheight=2

便利な機能

" バックスペースで削除
set backspace=indent,eol,start

" マウス操作有効
set mouse=a

" 自動インデント
set autoindent
set smartindent

" 改行時に自動でコメントアウトしない
autocmd FileType * setlocal formatoptions-=r
autocmd FileType * setlocal formatoptions-=o

🚀 クイックスタート

1 ~/.vimrc を作成
2 上の設定をコピペ
3 Vimを再起動
