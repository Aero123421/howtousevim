# How to use Vim

Vimの使い方をアニメーション豊富で分かりやすく解説するサイトです。

## サイト構成

- **ヒーローセクション** - Vimの紹介
- **基本概念** - 3つのモード（ノーマル・挿入・ビジュアル）を視覚化
- **カーソル移動デモ** - hjklと効率的な移動を実際に体験
- **インタラクティブチートシート** - カテゴリ別フィルタリング、コピー機能付き
- **実践シナリオ** - 関数の移動、一括置換、矩形編集の例
- **設定(.vimrc)** - おすすめ設定を紹介

## ファイル構成

- `index.html` - メインHTML
- `styles.css` - Vimグリーンテーマのスタイル
- `script.js` - インタラクティブ機能

## Vimの基本操作

### モード切り替え
- `i` - 挿入モードへ
- `Esc` - ノーマルモードへ
- `v` - ビジュアルモードへ

### カーソル移動（ノーマルモード）
- `h j k l` - 左・下・上・右
- `w` - 次の単語
- `b` - 前の単語
- `0` - 行頭
- `$` - 行末
- `gg` - 文書先頭
- `G` - 文書末尾

### 編集
- `x` - 1文字削除
- `dd` - 行削除
- `yy` - 行コピー
- `p` - ペースト
- `u` - 取り消し
- `Ctrl+r` - やり直し

### ファイル操作
- `:w` - 保存
- `:q` - 終了
- `:wq` - 保存して終了
- `:q!` - 保存せずに終了
